# Методичка

`06-room_thermostat` - это первый проект, который уже ближе к самостоятельной работе, чем к чистому повторению.

Что в нем есть:

- прямоугольный корпус
- круглый регулятор
- опциональный экран
- один электрический порт
- простое скрытие параметров

Задача этого проекта:

не столько прочитать новый материал, сколько попробовать самому узнавать знакомые паттерны из предыдущих примеров.

Что стоит сделать:

1. открыть `parameters.json`
2. самому объяснить, что делает каждая группа
3. открыть `main.lua`
4. попытаться без подсказки понять, где:
   - чтение параметров
   - `SetVisible()`
   - 3D-геометрия
   - 2D-символика
   - настройка порта
