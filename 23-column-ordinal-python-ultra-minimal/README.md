# 23-column-ordinal-python-ultra-minimal

Самая короткая версия нумерации колонн в Renga.

Что делает:
- подключается к уже открытой Renga;
- находит колонны;
- группирует их по `Марка группы`;
- записывает `1, 2, 3, ...` в `Порядковый номер в группе`.

Запуск:

```powershell
python .\23-column-ordinal-python-ultra-minimal\number_columns_min.py
```

Требуется:

```powershell
pip install pywin32
```
