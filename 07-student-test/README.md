# 07-student-test

Учебный проект простого подвесного пожарного спринклера по техническому паспорту.

Важно:
- это не точная заводская BIM-модель, а учебное приближение;
- основные размеры и пропорции взяты с изображения паспорта;
- мелкие вырезы на дефлекторе и сложные фаски пока упрощены.

Основные файлы:
- `parameters.json` - параметры стиля;
- `main.lua` - геометрия, символика и порт подключения;
- `build.ps1` - сборка шаблона;
- `metodichka.md` - разбор проекта.

Сборка:

```powershell
powershell -ExecutionPolicy Bypass -File .\07-student-test\build.ps1
```

Результат сборки:
- `07-student-test/sprinkler-head.rst`
