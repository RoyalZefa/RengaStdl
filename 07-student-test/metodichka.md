# Методичка

Этот проект показывает, как по техническому изображению собрать простой учебный STDL-объект.

Что здесь реализовано:
- параметризованный 3D-корпус;
- верхний дефлектор;
- простая 2D-символика для вида сверху;
- один трубный порт снизу;
- переключение параметров резьбы и условного диаметра через `SetVisible()`.

Что упрощено относительно паспорта:
- шестигранник сделан цилиндрическим поясом;
- стойки сделаны прямыми, без сложного профиля;
- дефлектор сделан сплошным диском без прорезей.

Как читать модель:
1. Нижний цилиндр - корпус с точкой подключения.
2. Верхний пояс - упрощенная гайка/посадочная часть.
3. Две стойки - рамка спринклера.
4. Узкий цилиндр - шток.
5. Верхний диск - дефлектор.

Если захотите довести проект ближе к паспорту, следующий хороший шаг:
- сделать шестигранный пояс через профиль и `Extrude(...)`;
- заменить прямые стойки на профиль по фронтальному контуру;
- усложнить 2D-символ сверху, добавив сектора дефлектора.
